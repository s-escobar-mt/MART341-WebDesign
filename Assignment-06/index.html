How to Draw a Cartoon Face

  1. First we start with a face shape

    2. This can be a circle, a square, a trapazoid etc.

    But for this example we will use a pear shape

  2. Next is the the nose

    3. Similar to face shape this can be any number of shapes

    In this example we use a rectangle

    And then erase the top most part

  3. Now onto the eyes

    4. Using the area above the nose a guide, place

    the eyes on either side of the nose.

    The style of the eyes is not important, but an easy

    eye to do is one that looks like a "T" with a rounded top.

    This example uses a variation of a "T" eye.

    Then add some curved lines above for eyebrows

  4. Onto the mouth

    5. Just a curved line will do for now

  5. Finally the rest of the head

    6. Here is where the hair and ears are added

Have fun creating some cartoon faces!

If interested in more realistic faces, try drawing from reference with Line of Action
